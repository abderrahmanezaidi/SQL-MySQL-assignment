# TP SQL – Voyageur Database

This repository contains a MySQL SQL script for the **Voyageur** database assignment.

The project includes:
- Creation of the database and tables
- Definition of primary and foreign keys
- Insertion of sample data
- SQL queries covering:
  - basic selections
  - sorting and filtering
  - joins between tables
  - aggregate functions
  - subqueries
  - set operations

The database models travelers, accommodations, stays, and activities.

## Files
- `voyageur.sql` – Complete MySQL script (schema, data, and queries)
